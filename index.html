Hallo ik ben Dani van der Mijl

Een Front-End webdeveloper in Nederland

Over mij

Mijn naam is Dani van der Mijl, een 19 jarige web front-end developer in Nederland. Welkom op mijn portfolio, waar ik laat zien waaraan ik heb gewerkt en waar ik nu sta kwa vaardigheden. De projecten op de site zijn gemaakt als onderdeel van schoolopdrachten, in samenwerking met bedrijven of als eigen projecten in mijn vrije tijd.

De talen die ik het beste beheers zijn HTML, SCSS, JS, CSS en JSON. Ook heb ik basiskennis van git/gitHub.

Na het halen van mijn diploma op het Van Der Meij College in Alkmaar, ben ik verder gegaan naar het Media College Amsterdam waar ik de opleiding Software Developer volg.

Mijn Skills

  • SCSS
  • JS
  • CSS
  • JSON
  • GIT

Kanao's Flip A Coin

Kanao's Flip A Coin is een website die ik zelf heb gemaakt. Het is gebaseerd op een scene uit de anime Demon Slayer. Het is idee achter de site is simpel, je een muntje kunt opgooien en de munt heeft een 50% kans om op 1 van de 2 kanten te landen, Tanjiro of Akaza.

Color Picker

Color Picker is een site gebaseerd op de bekende Colorhunt site. Het is een site die willekeurige kleuren genereert zodat je een mooie kleur kunt kiezen voor wat u maar nodig heeft. Deze site is volledig OOP.

Banky

Banky is een app gemaakt door middel van JSON data. In de app kan je tussen drie bank rekeningen verwisselen, je saldo zien en bekijken wie geld van je rekening heeft gehaald/erop gezet per rekening. De app is volledig OOP.
